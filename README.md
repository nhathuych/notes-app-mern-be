# notes-app-mern-be

## init project

Add package.json
```
npm init
```

Express(NodeJS framework), mongoose(ODM - Object Data Modeling) for MongoDB, nodemon(restart server automatically), ...
```
npm i express mongoose jsonwebtoken cors dotenv nodemon
```

## run source
```
npm start
```
